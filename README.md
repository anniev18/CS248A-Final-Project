# Cardinal3D: A Framework for Old Stanford CS248A Assignments

Cardinal3D is the starter codebase that was __previously used__ for programming assignments in Stanford CS248A: Computer Graphics.  Beginning with the [winter 2026 instantiation of the course](https://gfxcourses.stanford.edu/cs248a/winter26), the course moved on to a new set of assignments, but we are leaving these projects online for the general public to try.  Happy programming! 


Please see the following handouts for two programming assignments based on this code:

* [MeshEdit: Assignment 2 Handout](MeshEditHandout.md)
* [PathTracer: Assignment 3 Handout](PathTracerHandout.md)

We have created a [Wiki](https://stanford-cs248.github.io/Cardinal3D/) that describes the assignments in detail. On that page you will find all the details about how to use the starter code, what you need to implement, and tips/guidelines for implementing it.
